## Controles del juego
Para jugar a este juego tan solo se requieren dos teclas:
- _*Barra espaciadora*_: pulsar para saltar, mantener pulsado para rotar en el aire
- _*Q*_: pulsar para disparar bolas de nieve. Es importante notar que los disparos tienen cooldown, por lo que la cadencia de disparo es de 1 proyectil por segundo.

Funciones adicionales:
- _*G*_: activa/desactiva el generador de gravedad del clima.
- _*V*_: activa/desactiva el generador de viento del clima
- _*T*_: activa/desactiva el generador de torbellino alrededor del jugador
- _*Click del mouse + arrastrar*_: permite girar la vista
